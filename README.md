This is a simple runner game built using Python and Pygame.

## Features
- Dino character with jump mechanics
- Cactus and bird obstacles
- Collision detection
- Increasing difficulty over time
- Day and night cycle with sun, moon, and stars
- Score and high score system using file handling

## Controls
- SPACE / UP ARROW: Jump
- Close window to exit

## Requirements
- Python 3
- Pygame library
